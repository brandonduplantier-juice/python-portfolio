\# Global Happiness Dashboard



\## Live Dashboard

public.tableau.com/views/global\_happiness\_dashboard/GlobalHappinessDashboard2019



\## Overview

Interactive Tableau Public dashboard analyzing 2019 World Happiness Report

across 156 countries. Python used to merge and standardize 2015 and 2019

datasets before loading into Tableau.



\## Visuals

\- Choropleth world map colored by happiness score

\- Scatter plot with regression trend line (GDP vs happiness)

\- Top 20 countries bar chart

\- Happiness factors comparison (Freedom, Health, GDP, Generosity, Corruption)



\## Tools

\- Tableau Public Desktop

\- Python (pandas, data merging)

\- Dataset: World Happiness Report 2015 and 2019



\## Key Finding

Freedom has the highest average contribution to happiness at 0.87, followed

by Health at 0.66 and GDP at 0.40. Wealthiest nations are not always happiest.
